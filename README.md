### Prerequisites

- Node.js 16.x or later
- npm package manager

### Installation

1. Install dependencies:

```bash
npm install

```

2. Start the development server:

```bash
npm run dev

```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.
